1. HTML Meta Tags

Provide metadata about the HTML document, such as charset, viewport, and description.

<meta charset="UTF-8">
<meta name="viewport" 
content="width=device-width, 
initial-scale=1.0">
<meta name="description"
content="Learning HTML">

2. HTML Doctype

Define the document type and version of HTML used.

<!DOCTYPE html>

3. HTML Comments

Add comments to the code which will not be displayed in the browser.

<!-- This is a comment -->

4. HTML Responsive Design

Use meta tags and CSS media queries to create a responsive design.

<meta name="viewport" 
content="width=device-width, 
initial-scale=1.0">
<style>
@media (max-width: 600px) {
     body {
        background-color: lightblue;
     }
 }
</style>

5. HTML Data Attributes

Store custom data in HTML elements using data attributes.

<div data-user-id="123">User Profile</div>

6. HTML ARIA Roles for Accessibility

Enhance accessibility with ARIA roles for screen readers.

<div role="navigation">Navigation links</div>
<button aria-label="Close">Close</button>

7. HTML5 New Elements (<article>, <section>, <aside>...)

HTML5 introduces new semantic elements for better page structure.

<article>This is an article</article>
<section>This is a section</section>
<aside>This is an aside</aside>

8. HTML5 Local Storage

Store data locally on the user's device using localStorage.

localStorage.setItem("username", "JohnDoe");
let username = 
localStorage.getItem("username");

9. HTML5 Web Workers

Run scripts in the background using Web Workers.

const worker = new Worker('worker.js');
worker.postMessage("start");

worker.onmessage = function(e) {
    console.log(e.data);
};
