1. HTML Tables

Organize data into rows and columns using tables.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

2. HTML Forms

Collect user input through forms.

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

3. HTML Input Types

Different types of input fields for forms.

<input type="text" 
placeholder="Enter text">
<input type="password" 
placeholder="Enter password">
<input type="email" 
placeholder="Enter email">

4. HTML Audio and Video

Embed multimedia content like audio and video.

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls>
<source src="video.mp4" type="video/mp4">
</video>

5. HTML Semantic Elements

Use meaningful tags like <header>, <footer>, <article> for better structure.

<header>
    <h1>Site Header</h1>
</header>

<article>
    <p>This is an article.</p>
</article>

<footer>
    <p>Site Footer</p>
</footer>

6. HTML Block and Inline Elements

Understanding block-level and inline elements.

<div>This is a block-level element.</div>
<span>This is an inline element.</span>

7. HTML Entities

Use HTML entities for special characters like &, <, >, etc.

<p>This is a <b>bold</b> text.</p>
<p>Use & for the ampersand symbol.</p>
<p>Use <&gt;< for 
less than and greater than symbols.</p>

8. HTML Iframes

Embed another HTML page within the current page using iframes.

<iframe src="https://example.com" width="600" 
height="400"></iframe>

9. HTML Global Attributes

Attributes that can be used on any HTML element, like <class>, <id>, etc.

<div id="container" 
class="main-container">Content here</div>

10. HTML Best Practices

Write clean, readable, and maintainable code.

<!-- Use meaningful class and id names -->
<div id="main-container">
    <h1>Title</h1>
</div>
