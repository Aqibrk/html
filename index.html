HTML Learning Guide Basic

Master HTML concepts with practical examples and resources

1. HTML Introduction

An overview of HTML and its purpose in creating web pages.

<!DOCTYPE html>
<html>
<head>
    <title>HTML Introduction</title>
</head>
<body>
    <h1>Welcome to HTML!</h1>
    <p>This is a basic HTML page.</p>
</body>
</html>

2. HTML Syntax

The basic structure of an HTML document.

<!DOCTYPE html>
<html>
<head>
    <title>HTML Syntax</title>
</head>
<body>
    <h1>Heading</h1>
    <p>Paragraph text.</p>
</body>
</html>

3. HTML Elements

HTML elements are the building blocks of web pages, defined by tags.

<p>This is a paragraph.</p>
<a href="https://example.com">
This is a link.</a>

4. HTML Attributes

Attributes provide additional information about an element.

<a href="https://example.com">Open Link</a>
<img src="image.jpg" alt="Sample Image">

5. HTML Headings

Six levels of headings from <h1> to <h6>.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

6. HTML Paragraphs

Paragraphs are used to group text content into blocks.

<p>This is a paragraph of text.</p>
<p>This is another paragraph.</p>

7. HTML Lists

Create ordered and unordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<ol>
    <li>First</li>
    <li>Second</li>
</ol>

8. HTML Links

Create hyperlinks to other pages or resources.

<a href="https://example.com"> Example</a>

9. HTML Images

Embed images using the <img> tag.

<img src="image.jpg" alt="Des of Image">

10. HTML Div and Span

Generic containers for structuring content.

<div>
    <p>This is a block-level element.</p>
</div>
<span>This is an inline element.</span>
